---
name: packing-list
description: 旅行先・期間・季節・旅のスタイルを元に、忘れ物なしの持ち物リストを生成します。
---

## 手順

1. 以下の情報をユーザーに確認する
   - 旅行先（国内 / 海外）
   - 旅行期間（日数）
   - 季節・天気予報
   - 旅のスタイル（観光 / アウトドア / ビジネス など）
2. カテゴリ別の持ち物リストを生成する

## 出力フォーマット

```markdown
# 持ち物リスト（〇〇旅行 / X 泊 Y 日）

## 必需品
- [ ] パスポート / 身分証明書
- [ ] 財布・現金・クレジットカード
- [ ] スマートフォン・充電器
- [ ] 常備薬

## 衣類
- [ ] 着替え（X 日分）
- [ ] 下着・靴下（X 日分）
- [ ] 羽織もの（季節に応じて）

## 洗面・衛生用品
- [ ] 歯ブラシ・歯磨き粉
- [ ] シャンプー・ボディソープ（宿泊先になければ）
- [ ] 日焼け止め

## ガジェット
- [ ] カメラ・予備バッテリー
- [ ] モバイルバッテリー
- [ ] イヤホン

## その他（旅行先・スタイルに応じて）
- [ ] （自動生成）
```
