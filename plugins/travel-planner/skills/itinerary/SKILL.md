---
name: itinerary
description: 行き先と日数を指定するだけで、観光スポット・グルメ・移動手段を含むタイムテーブル形式の旅程表を生成します。
---

## 手順

1. ユーザーから行き先と日数を受け取る
2. 代表的な観光スポット・グルメ情報を元に旅程を組み立てる
3. タイムテーブル形式で出力する

## 出力フォーマット

```markdown
# 〇〇 旅程表（X 泊 Y 日）

| 時間 | 内容 | 場所 | メモ |
|------|------|------|------|
| 09:00 | チェックアウト | 〇〇ホテル | |
| 10:00 | 〇〇観光 | 〇〇 | 所要 1 時間 |
| 12:00 | ランチ | 〇〇レストラン | 名物: 〇〇 |
| 14:00 | 〇〇観光 | 〇〇 | 入場料: 〇〇円 |
| 17:00 | 移動 | 〇〇駅 → 〇〇駅 | 電車 XX 分 |
| 18:00 | チェックイン | 〇〇ホテル | |
| 19:00 | 夕食 | 〇〇 | 予算: 〇〇円/人 |
```

## 使用例

```
京都に 2 泊 3 日で行きます。itinerary を作って
```
