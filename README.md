# github-copilot-cli-plugin-sample

GitHub Copilot CLI のプラグイン・マーケットプレイスのサンプルリポジトリです。

## マーケットプレイスとして使う

```bash
copilot plugin marketplace add apc-nonoshita/github-copilot-cli-plugin-sample
```

```bash
copilot plugin install travel-planner@plugin-sample
```

## プラグイン一覧

| プラグイン | 説明 |
|-----------|------|
| [travel-planner](./plugins/travel-planner/) | 旅行・おでかけの計画を立てるためのプラグインサンプル |
